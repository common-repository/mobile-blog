== Description ==

O plugin Mobile Blog permite que seu blog seja acessado através de
dispositivos móveis (celulares, smartphones e todos esses “brinquedinhos”).

Ele tem a capacidade de redimensionar as imagens para 150×150 pixels,
diminuindo o tempo de carregamento e o valor gasto (R$) para acessar
as páginas. Além de retornar um HTML bem “compacto”. Criei ele tentando
seguir os padrões WAP, para ser compatível com um número maior de
dispositivos.

Por enquanto ele faz basicamente isso. Mas logo terá as opções de exibir
comentários dos posts, uma possível integração com redes sociais e
algumas outras melhorias.

Desafios para as próximas versões:

1. Exibir os comentários
1. Mostrar o Gravatar
1. Permitir comentários
1. Acesso ao painel administrativo
1. Adicionar mais temas, e permitir a personalização pelo painel

Ele identifica automaticamente o dispositivo. Não precisa criar nenhuma
página, URL, nem nada do tipo. É só acessar o endereço do seu blog no
navegador do seu celular, por exemplo.Mas para garantir que o plugin foi
corretamente instalado, abra o seu blog adicionando ?mb na URL.
Exemplo: http://techlive.org?mb.

== Installation ==

Esta seção explica como instalar o plugin:

1. Faça download, extraia o .zip e faça upload para a pasta /wp-content/plugins do seu Wordpress.
2. Dê um chmod 777 no diretório /thumbs/cache.
3. Ative o plugin.
4. Pronto!
